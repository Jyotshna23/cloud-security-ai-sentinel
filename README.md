🚨 Problem Statement
Every day, thousands of cyber attacks target Indian companies, hospitals, banks, and government systems. Security teams manually review hundreds of alerts — missing critical threats due to human error and fatigue. This AI-powered sentinel automatically analyzes every security event in real-time, classifying threats and recommending immediate actions before damage occurs.
⚡ What This Project Does
Cloud Security AI Sentinel uses Google Gemini AI to analyze cloud security events and classify them by severity with actionable recommendations, running automatically every 6 hours via GitHub Actions.

✅ Analyzes security events using Gemini AI
✅ Classifies threats: CRITICAL, HIGH, MEDIUM, LOW
✅ Provides instant actionable recommendations
✅ Runs automatically every 6 hours
✅ Generates summary reports

🌍 Real World Impact
🏥 Healthcare — Detects unauthorized access to patient records before data breach occurs.
🏦 Banking — Identifies suspicious login patterns and unusual data downloads.
🏛️ Government — Monitors configuration changes indicating insider threats.
🚀 Startups — Enterprise-grade security monitoring at zero cost.
